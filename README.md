# supply-chain-analytics


## Problem Statement
Analyzed 180,000+ supply chain transactions to identify delivery 
bottlenecks, revenue trends, and shipping performance.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI

## Key KPIs Tracked
- Late Delivery Rate
- Revenue by Product Category
- On-Time Fulfillment by Shipping Mode
- Monthly Order Volume Trend

## Key Findings
- XX% of orders face late delivery risk
- [Shipping Mode] has the highest on-time rate
- [Category] drives XX% of total revenue

## How to Run
1. Download dataset from Kaggle (link)
2. Run `notebooks/01_eda_cleaning.ipynb`
3. Load processed CSV into SQL
4. Open `dashboard/supply_chain.pbix` in Power BI
